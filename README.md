# 💰 Personalized Financial Advisory System Using AI

## 📌 Overview

The **Personalized Financial Advisory System** is an AI-powered application that helps individuals manage their finances effectively. It analyzes user data such as income, expenses, financial goals, and risk appetite to generate personalized recommendations for budgeting, saving, and investing.

This system aims to make financial planning simple, accessible, and data-driven.

---

## 🚀 Features

* 📊 Income and Expense Analysis
* 💡 Personalized Financial Recommendations
* 📈 Investment Suggestions
* 🧾 Smart Budget Planning
* 🎯 Goal-Based Financial Planning
* 🤖 Machine Learning Insights

---

## 🛠️ Tech Stack

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Personalized-Financial-Advisory-System.git
cd Personalized-Financial-Advisory-System
```

### 2. Create Virtual Environment (Recommended)

```bash
python -m venv venv
```

Activate environment:

* Windows:

```bash
venv\Scripts\activate
```

* Linux/Mac:

```bash
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
python app.py
```

---

## 🧠 How It Works

1. User inputs financial details (income, expenses, goals)
2. Data is processed and cleaned
3. Machine Learning models analyze patterns
4. System generates personalized financial advice

---

## 📌 Example Input

```
Income: 50000
Expenses: 30000
Savings Goal: Buy a car
Risk Level: Medium
```

## 📌 Example Output

```
- Save at least 20% of your income monthly
- Reduce unnecessary expenses
- Invest in balanced mutual funds
```
